Osu

Telegram-бот на библиотеке aiogram, предназначенного для вывода информации о аккаунте в игре Osu! через API. Бот предоставляет статистику аккаунта, информацию о последних сыгранных картах и топовых пройденных картах. Этот проект демонстрирует мою способность использовать aiogram для взаимодействия с Telegram API и получения данных из API Osu! для предоставления пользователям актуальной игровой информации.
